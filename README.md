# 📋 Sistema de Cadastro de Usuários – Java Console App

Este projeto é um sistema simples de **cadastro de usuários em Java**, utilizando o console para entrada de dados.

Ele permite:

- Cadastrar novos usuários (nome e idade)
- Listar os usuários cadastrados
- Encerrar o programa

---

## ☕ Feito com Java

Java é uma linguagem robusta, multiplataforma, e orientada a objetos.  
Este projeto demonstra o uso de:

- Classes e Objetos
- Métodos
- Condicionais (`switch`)
- Entrada de dados com `Scanner`

---

## 📜 Código principal (`Main.java`)

### 📌 O que ele faz:

1. Mostra um menu com 3 opções:
   - Cadastrar novo usuário
   - Listar usuários
   - Sair

2. Coleta dados do usuário via `Scanner`

3. Usa `main(new String[]{""})` para **reiniciar o programa** ao invés de usar loops (alternativa simples, porém não recomendada em projetos grandes).

---

## 🧠 O que foi aprendido neste projeto

- Como usar a classe `Scanner` para ler dados do usuário
- Criar e usar uma classe separada (`Users`) para gerenciar dados
- Implementar um menu com `switch`
- Princípios de **modularidade** e **responsabilidade de classes**
